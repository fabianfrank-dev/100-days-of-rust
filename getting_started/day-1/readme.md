# 🦀 Day 1: Rust Basics

Today I started learning Rust! Here’s what I covered:

- **Project setup**: `cargo new` to create a new Rust project.
- **Variables**: `let` for immutable, `let mut` for mutable values.
- **User input**: `std::io::stdin()` to read input, `.expect()` for error handling.
- **Compiling**: `cargo run` or `rustc main.rs && ./main`.

### Scripts
1. **Hello World** – Basic output with `println!`.
2. **User Input** – Reading and printing user input.
3. **Calculator** – Simple addition of two numbers.

Each script is in its own folder under `day-01/`.
